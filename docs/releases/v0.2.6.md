## Summary

- Adds a signed GitHub Pages-backed APT repository so Linux users can install and update Load Rift with `sudo apt install load-rift`.
- Publishes an APT repository setup package and `install-apt-repo.sh` helper with each release.
- Documents macOS DMG installs, Linux APT installs, and direct-download fallback assets in the README.
- Adds Linux AppStream metadata to the `.deb` package for better desktop software-center integration.

## Linux APT Install

Enable the Load Rift APT repository once:

```bash
bash <(curl -fsSL https://github.com/ddv1982/load-rift/releases/latest/download/install-apt-repo.sh)
```

Then install or update normally:

```bash
sudo apt update
sudo apt install load-rift
```
