## Summary

- Publishes Load Rift 0.2.16 with app metadata aligned across npm, Cargo, and Tauri packaging.
- Improves keyboard navigation visibility with explicit focus rings for the dark custom controls.
- Honors reduced-motion preferences and relaxes small-screen monitor/result panel heights to reduce nested scrolling pressure.
- Disables empty-state Reset and report export actions until a collection or completed result makes them actionable.
- Refreshes frontend dependencies to the newest npm-published versions available for this release line.

## Linux APT Install

Enable the Load Rift APT repository once:

```bash
bash <(curl -fsSL https://github.com/ddv1982/load-rift/releases/latest/download/install-apt-repo.sh)
```

Then install or update normally:

```bash
sudo apt update
sudo apt install load-rift
```

The setup script authenticates the repository setup package checksum with a detached GPG signature and a release-pinned signer fingerprint before installing the repository configuration.
