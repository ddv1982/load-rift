## Summary

- Hardens k6 run temp artifacts so spawned k6 processes keep access to their generated `script.js`, `summary.json`, and `metrics.json` for the full run lifecycle.
- Adds child-PID artifact markers, conservative stale cleanup, and strict marker/shape validation before Load Rift deletes old temp directories.
- Redacts local artifact paths from user-visible fallback diagnostics by default while keeping opt-in debug preservation available with `LOADRIFT_PRESERVE_K6_ARTIFACTS=true`.
- Enforces bundled-k6 regression coverage in CI with `LOADRIFT_REQUIRE_BUNDLED_K6_TESTS=true`.
- Bumps Load Rift metadata to version 0.2.10.

## Linux APT Install

Enable the Load Rift APT repository once:

```bash
bash <(curl -fsSL https://github.com/ddv1982/load-rift/releases/latest/download/install-apt-repo.sh)
```

Then install or update normally:

```bash
sudo apt update
sudo apt install load-rift
```

The setup script authenticates the repository setup package checksum with a detached GPG signature and a release-pinned signer fingerprint before installing the repository configuration.

## Lore

Constraint: Release notes are read by the GitHub Actions release workflow from `docs/releases/v0.2.10.md`.
Rejected: Deleting stale `loadrift-*` directories based only on age and name | it could remove artifacts still needed by a live or orphaned k6 process.
Rejected: Showing full local temp paths in normal fallback output | it leaks usernames/local paths into reports when debug artifact preservation was not requested.
Rejected: Letting bundled-k6 smoke tests silently skip in CI | it would miss the regression class this release is meant to prevent.
Confidence: high
Scope-risk: moderate
Directive: Keep k6 artifact cleanup conservative; prefer leaving stale temp files behind over deleting files that an uncertain k6 child might still need.
Tested: `cargo fmt --manifest-path src-tauri/Cargo.toml --check`; `LOADRIFT_REQUIRE_BUNDLED_K6_TESTS=true cargo test --manifest-path src-tauri/Cargo.toml k6::tests --lib`; `cargo test --manifest-path src-tauri/Cargo.toml importing:: --lib`; `cargo check --manifest-path src-tauri/Cargo.toml`; `npm run typecheck`; `npm run lint`; `npm run build`; `git diff --check`; focused RepoPrompt post-fix review.
