## Summary

- Updates the bundled Grafana k6 sidecar from v1.6.1 to v2.0.0 with refreshed release-asset checksums.
- Updates npm, Tauri, and Rust dependencies to their latest stable compatible versions.
- Refreshes bundled third-party licensing notices to identify k6 v2.0.0 as AGPL-3.0-only and links the exact corresponding source.
- Keeps the Windows frontend build target at Chrome 105 while raising the WebKit target to Safari 15 for compatibility with the latest esbuild toolchain.
- Bumps Load Rift metadata to version 0.2.8.

## Linux APT Install

Enable the Load Rift APT repository once:

```bash
bash <(curl -fsSL https://github.com/ddv1982/load-rift/releases/latest/download/install-apt-repo.sh)
```

Then install or update normally:

```bash
sudo apt update
sudo apt install load-rift
```

The setup script authenticates the repository setup package checksum with a detached GPG signature and a release-pinned signer fingerprint before installing the repository configuration.
