## Summary

- Added first-class weighted request mix support so selected requests can carry relative importance during a run.
- Hardened the built-in weighted mode into a deterministic started-iteration schedule and aligned weight `0` as an exclusion signal across the UI, persistence, and runtime.
- Kept advanced k6 scenarios as the precise override path for stricter workload-shape control.
