## Summary

- Fixes k6 live-metrics fallback results so non-zero or unknown k6 exits are reported as failed instead of passed.
- Preserves the primary k6 stderr/error message in the UI when a same-run generic `k6:error` event arrives after completion.
- Tightens the bundled k6 private-temp smoke test to exercise both `--summary-export` and production-style `--out json=<metrics_path>` output.
- Bumps Load Rift metadata to version 0.2.9.

## Linux APT Install

Enable the Load Rift APT repository once:

```bash
bash <(curl -fsSL https://github.com/ddv1982/load-rift/releases/latest/download/install-apt-repo.sh)
```

Then install or update normally:

```bash
sudo apt update
sudo apt install load-rift
```

The setup script authenticates the repository setup package checksum with a detached GPG signature and a release-pinned signer fingerprint before installing the repository configuration.
